## r/chrome or r/browsers

**Title:** I got tired of my tab bar being one screaming row, so I built a little "tab librarian"

**Body:**
My tab bar was a wall of identical favicons and I could never find anything. So I
made **shelf** — a tiny extension that auto-groups your tabs onto coloured "shelves"
by domain (and your own rules), and collapses the ones you're not using.

- Groups only loose tabs — it never touches groups you made by hand.
- Opens a link → the tab stays put; it shelves it once you move away (no jumping).
- Local only, no account, no cloud. Manifest V3, open source (MIT).

Works in Chrome / Brave / Edge. Built it for myself, figured I'd share. Feedback
welcome. LINK

## Hacker News (Show HN)

**Title:** Show HN: shelf – a tiny tab librarian that auto-shelves your browser tabs

**Body:**
shelf auto-groups browser tabs onto coloured shelves by domain or your own rules,
and collapses the inactive ones. I built it because my tab bar was one unreadable
screaming row.

Notable bits for this crowd:
- **No build step.** Vanilla JS, ES modules — what's in the repo is what runs.
- **Zero dependencies**, zero telemetry, nothing leaves the browser.
- The icon and the README demo animation are both generated by hand-rolled
  PNG/APNG encoders on Node's built-in zlib (no image libs) — `scripts/`.
- It groups a tab only when you *leave* it, so opening a link never yanks the tab
  you're reading into a faraway group. It also respects groups you made manually.

Manifest V3, Chromium (Chrome/Brave/Edge); Firefox is waiting on a stable tabGroups
API. MIT. LINK
